# Poker for Toddler — Card Matching Learning Game

## Overview
A colorful, gender-neutral poker game designed to teach children aged **4–8** pattern recognition, card matching, comparison, and basic strategy — all through fun, gamified play. No gambling language, no real money — just learning wrapped in confetti and bright colors.

### What Kids Learn
- **Pattern recognition:** Spotting pairs (two matching numbers)
- **Classification:** Grouping cards by color (red vs black) and suit (Hearts, Diamonds, Clubs, Spades)
- **Comparison:** Understanding which hand is "better" through a simple point system
- **Decision-making:** Choosing whether to swap a card or keep what you have
- **Turn-taking & sportsmanship:** Playing against a dealer, winning and losing gracefully

## Tech Stack
- **Frontend:** React 18 + Vite 6
- **Styling:** Inline styles (no CSS framework) — intentional for simplicity and animation control
- **PWA:** Custom service worker (`public/sw.js`) + web app manifest — no vite-plugin-pwa
- **iOS Native:** Capacitor 6 (`@capacitor/core` + `@capacitor/ios`, wraps the web build into a native iOS shell)
- **Font:** Fredoka One + Nunito (Google Fonts, loaded via CSS `@import` in the `<style>` tag)
- **Hosting:** Vercel (auto-deploys from `main`)

## Architecture Decisions
- **Single-file component (`App.jsx`):** The entire game lives in one file. This is intentional — the game is small, self-contained, and benefits from locality. Do NOT split into multiple files unless the file exceeds ~600 lines.
- **Inline styles over CSS:** Keeps animations co-located with components. Global keyframes live in a `<style>` tag inside `App`.
- **No state management library:** `useState` + `useCallback` is sufficient. Do NOT add Redux/Zustand/etc.
- **No router:** Single-screen app. No routing needed.

## Game Rules (Simplified Poker for Kids)
- **Deck:** Standard 52 cards, reshuffled each deal
- **Deal:** Player and Dealer each get 2 cards
- **Hands (simplified, 3 tiers):**
  - **Pair** (3 pts) — two cards with the same rank (e.g. 5+5)
  - **Color Match** (2 pts) — both cards same color (both red or both black)
  - **High Card** (1 pt) — no match, highest card value counts
- **Swap phase:** Player may swap 1 card for a new one from the deck, or keep both
- **Showdown:** Dealer cards revealed, hands compared, highest points wins. Ties broken by highest individual card value.
- **No bluffing, no betting, no chips** — purely educational pattern matching

## Game Flow (Phases)
1. `intro` — Welcome screen with rules explanation and "Deal Cards!" button
2. `pick` — Player sees their 2 cards, dealer cards hidden. Player can tap a card to select it, then "Swap Card" or "Keep Both!"
3. `swap` — Cards are locked in, "Showdown!" button appears
4. `reveal` — Dealer cards flip over with a thinking animation
5. `result` — Winner announced with hand comparison, confetti on win, "Play Again!" button

## Conventions
- Use `'Fredoka One', cursive` for display text, `Nunito, sans-serif` for body
- Color palette: #FF4757 (red), #FFA502 (orange), #2ED573 (green), #1E90FF (blue), #6c5ce7 (purple)
- All buttons use the `.btn` class for shared hover/active transitions
- Animations: prefer CSS @keyframes over JS animation libraries
- Touch targets: minimum 44x44px for all interactive elements (Apple HIG)
- Keep everything child-safe: no gambling language, no real money references
- Gender-neutral language and imagery throughout — no gendered pronouns or characters
- Encouraging tone: celebrate wins enthusiastically, soften losses ("Dealer wins this time!" not "You lost!")
- Age-appropriate vocabulary: short sentences, familiar words, visual cues alongside text

## Design Philosophy
The game is designed around the principle that **kids learn best when they're having fun**. Every interaction should feel rewarding:
- Correct patterns are highlighted with color and animation
- The point system is visible and explained (not hidden)
- The learning guide is always one tap away
- Confetti and bouncing emojis make wins feel special
- Losses are gentle and immediately offer another round
- Cards are large, colorful, and satisfying to tap
